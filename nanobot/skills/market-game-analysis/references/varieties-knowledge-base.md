## SC 原油

### 定价决定性力量

| 力量 | 说明 |
|------|------|
| **OPEC+ 减产联盟** | 沙特的减产意愿是核心，决定全球原油供应预期 |
| **美国页岩油** | 边际供应者，页岩油成本（约 $50-60/桶）是油价底部 |
| **中国需求** | 2025 年后中国新能源车替代汽柴油消费，需求增速放缓 |
| **地缘冲突** | 霍尔木兹海峡/红海封锁直接影响运输溢价 |

### 什么条件下会赢

- **OPEC+ 减产 + 地缘紧张** → 油价涨
- **美国增产 + 全球需求弱** → 油价跌
- **地缘溢价**：霍尔木兹封闭时，SC 相对 WTI/布伦特溢价扩大（亚太到岸价溢价）
- **SC 逼仓条件**：近月合约持仓量 >> 可交割库存，且中东局势紧张

### 历史量价规律

- 地缘冲突驱动型行情：涨幅大（+5% 以上）、V/OI 快速攀升至 2 以上，持续性取决于地缘持续时间
- 库存数据驱动型行情：通常涨幅在 1-3%，V/OI 在 1-1.5
- 换月前后：持仓量从近月向远月转移，成交量放大

### 容易被操纵的特征

- 国内 SC 相对国际油价有时滞，内外盘联动时可能产生套利空间
- 低成交合约（SC_TAS 等）成交量极低，不参与

## AU 黄金

### 定价决定性力量

| 力量 | 说明 |
|------|------|
| **实际利率** | 核心驱动，实际利率下行 → 黄金涨（机会成本降低） |
| **美元指数** | 负相关，美元强 → 黄金跌 |
| **避险情绪** | 地缘冲突/金融危机时避险买盘推动 |
| **央行购金** | 2022 年后各国央行（尤其中国/印度）增持黄金 |

### 什么条件下会赢

- **实际利率下行**（降息周期）→ 黄金涨
- **美元走弱** → 黄金涨
- **地缘冲突爆发** → 避险买盘推动
- **央行持续购金** → 底部支撑强

### 历史量价规律

- 避险驱动行情：价格急涨，成交量放大，但波动率极高
- 实际利率驱动行情：趋势性强，持续时间长（数周到数月）
- 美元和黄金常常"脱钩"——传统模型失效时，可能是央行/机构资金在主导

### 容易被操纵的特征

- COMEX 期货持仓量极高，大型机构可以左右短期价格
- 国内 AU 跟随 COMEX + 人民币汇率双重影响

## AG 白银

### 定价决定性力量

| 力量 | 说明 |
|------|------|
| **黄金联动** | 与黄金相关性高，是黄金的"杠杆版" |
| **工业需求** | 光伏/电子行业需求，占总需求约 50% |
| **金银比** | 金银比极端值（80+）时，白银有均值回归动力 |

### 什么条件下会赢

- **黄金大涨** → 白银跟涨（弹性更大）
- **工业需求回暖** → 白银基本面改善
- **金银比极端高** → 白银相对黄金被低估，有补涨动力

### 历史量价规律

- 涨幅通常大于黄金（同向波动时）
- 工业属性强时，与黄金走势可能短暂背离
- **V/OI > 1.5 = 极端警告信号，不是趋势确认**
- **V/OI > 2.5 + 放量下跌 = 有效做空机会**（资金主动做空，行情未完）
- **V/OI > 5 = 极端情绪顶，可能随时反弹**（2026-07-17 实盘：SN 锡 V/OI=5.74，放量下跌，次日未追空正确）

## RB 螺纹钢

### 定价决定性力量

| 力量 | 说明 |
|------|------|
| **钢厂利润** | 核心枢纽，利润高 → 钢厂增供 → 价格压力 |
| **房地产需求** | 螺纹钢最大下游，地产下行周期钢价承压 |
| **政策预期** | 基建刺激/限产政策可以短期扭转供需 |
| **冬储博弈** | 每年 11-12 月贸易商冬储行为影响短期价格 |

### 什么条件下会赢

- **地产需求回暖 + 钢厂利润低** → 钢厂联合挺价
- **政策刺激基建** → 需求预期改善，价格涨
- **钢厂大规模检修/限产** → 供应收缩，价格涨

### 历史量价规律

- 冬储期间：贸易商行为主导，价格区间震荡
- 旺季（3-4 月）：需求启动，价格往往有一波上涨
- 供过于求时：价格跌破钢厂成本，触发减产

### 容易被操纵的特征

- 成交额低的月份合约（远月合约）容易被操纵
- 冬储博弈期间价格波动幅度大

## JM 焦煤

### 定价决定性力量

| 力量 | 说明 |
|------|------|
| **钢厂压价意愿** | 钢厂利润低时联合压价，焦煤价格跟跌 |
| **澳煤进口政策** | 澳煤进口恢复/受限直接冲击国内焦煤价格 |
| **蒙古进口煤** | 甘其毛都口岸通关量影响供应 |
| **安全监管** | 山西/陕西煤矿安监影响国内产量 |

### 什么条件下会赢

- **钢厂利润高** → 压价意愿弱，焦煤价格可以涨
- **澳煤进口受限** → 国内焦煤供需偏紧
- **煤矿安全监管收紧** → 国产供应收缩

### 历史量价规律

- 地缘溢价驱动（2026年3月）：焦煤涨停 11%+，是情绪驱动+基本面共振
- 钢厂压价驱动：价格下跌持续性较强，直到钢厂利润改善
- V/OI 极端时（>2）：往往是情绪顶点，不是趋势确认

## I 铁矿石

### 定价决定性力量

| 力量 | 说明 |
|------|------|
| **四大矿山发货量** | 淡水河谷/力拓/BHP/FMG 决定全球铁矿供应 |
| **钢厂利润** | 利润高 → 钢厂用好矿增多 → 高品矿溢价 |
| **港口库存** | 库存高 → 价格压力；库存低 → 供应紧张 |

### 什么条件下会赢

- **矿山发货受阻**（飓风/事故）→ 价格急涨
- **钢厂利润高** → 高品矿需求好，价格涨
- **港口库存持续下降** → 供需偏紧

### 历史量价规律

- 铁矿石涨幅通常领先螺纹钢，是黑色系的风向标
- 发货量数据（每周）影响短期价格
- 期货价格和现货价格（普氏指数）联动紧密

## MA 甲醇

### 定价决定性力量

| 力量 | 说明 |
|------|------|
| **煤炭成本** | 国内甲醇以煤制为主，煤价是核心成本 |
| **MTO 装置利润** | 烯烃工厂是最大下游，MTO 利润决定采购意愿 |
| **进口量** | 进口依存度约 30%，进口价格影响国内定价 |
| **库存周期** | 6-9 月累库，9-12 月去库 |

### 什么条件下会赢

- **煤价上涨** → 甲醇成本支撑，价格涨
- **MTO 装置利润好** → 需求强劲，价格涨
- **进口量下降** → 沿海供应偏紧

### 历史量价规律

- 成本推动型：煤价涨 → 甲醇涨（传导有时滞）
- MTO 停车：烯烃工厂停车时甲醇需求骤降，价格急跌
- 进口缺口故事：进口减少+沿海库存低时，容易出现逼仓行情

## SN 锡

### 定价决定性力量

| 力量 | 说明 |
|------|------|
| **缅甸佤邦矿端** | 全球最大锡矿之一，政策/封关直接影响供应 |
| **印尼出口政策** | 印尼是最大锡出口国，出口配额影响全球供应 |
| **电子行业需求** | 锡最大下游是电子焊料，芯片周期是核心 |

### 什么条件下会赢

- **缅甸矿端供应收紧** → 锡价涨
- **印尼限制出口** → 锡价涨

- 缅甸/印尼政策驱动型：涨幅大（+5% 以上），持续性取决于政策持续时间

### 容易被操纵的特征

- 成交额相对铜、铝偏小，日内波动大

## SR 白糖

### 定价决定性力量

| 力量 | 说明 |
|------|------|
| **原糖成本** | 国际糖价（纽约原糖）是国内白糖定价的锚 |
| **国储节奏** | 国家收储/抛储直接调节市场流通量 |
| **进口配额** | 进口政策影响国内供应松紧 |
| **石油价格** | 甘蔗可以制乙醇，石油涨 → 乙醇需求高 → 甘蔗转向能源，糖供应降 |

### 什么条件下会赢

- **国储抛储** → 供应增加，价格压力
- **国储收储** → 供应减少，价格支撑
- **石油大涨** → 甘蔗制乙醇比例提高，糖供应减少
- **进口配额收紧** → 国内供应偏紧

### 历史量价规律

- 政策驱动型行情：抛储/收储政策出台后价格有明显反应
- 厄尔尼诺天气：影响巴西/泰国甘蔗产量，影响国际糖价
- 牛熊周期约 5-7 年，供需错配驱动大趋势

## P 棕榈油

### 定价决定性力量

| 力量 | 说明 |
|------|------|
| **马来西亚产量** | 全球最大棕榈油生产国，产量是核心 |

- **豆油大涨** → 棕榈油跟涨

### 历史量价规律

- 季节性明显：每年 11-2 月增产季价格承压，3-10 月减产季价格偏强
- 厄尔尼诺天气影响马来西亚产量，滞后 6-12 个月显现
- 与豆油、菜籽油联动紧密

## M 豆粕

### 定价决定性力量

| 力量 | 说明 |
|------|------|
| **CBOT 大豆** | 国内豆粕跟随 CBOT 大豆定价，进口大豆成本是核心 |
| **生猪存栏** | 豆粕是饲料，生猪存栏决定需求 |
| **油厂压榨量** | 大豆压榨产出豆粕，压榨量决定供应 |
| **贸易政策** | 关税政策（中美贸易）影响大豆进口成本 |

### 什么条件下会赢

- **CBOT 大豆上涨** → 进口成本涨，豆粕跟涨
- **生猪存栏高** → 饲料需求好，豆粕涨
- **油厂检修/停机** → 豆粕供应减少，价格涨
- **中美贸易摩擦** → 大豆关税提高，豆粕成本涨

### 历史量价规律

- CBOT 大豆是领先指标，国内连粕跟随
- 生猪存栏周期滞后影响豆粕需求
- 油厂开机率（每周）影响短期供应
- 豆粕与豆油存在跷跷板效应（压榨利润）

## LH 生猪

### 定价决定性力量

| 力量 | 说明 |
|------|------|
| **能繁母猪存栏** | 决定 10 个月后生猪出栏量，是供应的领先指标 |
| **养殖利润** | 利润高 → 养大猪/压栏 → 短期供应减少 |
| **收抛储政策** | 国家收储托底，抛储压价 |
| **疫病** | 非洲猪瘟等疫病导致产能去化 |

### 什么条件下会赢

- **能繁母猪存栏下降** → 10 个月后供应减少，价格涨
- **养殖利润高** → 压栏/二次育肥，短期供应减少
- **国家收储** → 需求托底，价格支撑

### 历史量价规律

- 猪周期约 4 年，能繁母猪存栏是核心领先指标
- 体重（出栏均重）是短期供需的灵敏指标
- 盘面呈现高贴水结构（期货反映未来价格预期）

## C 玉米

### 定价决定性力量

| 力量 | 说明 |
|------|------|
| **国储拍卖** | 国内玉米主要来源，拍卖节奏影响供应 |
| **进口配额** | 进口玉米受配额限制，配额外关税高 |

### 什么条件下会赢

- **国储拍卖减少** → 供应收紧，价格涨
- **养殖存栏高** → 饲料需求好，价格涨
- **产区干旱** → 减产预期，价格涨

### 历史量价规律

- 国储库存是价格的重要锚点
- 进口玉米到港成本（含关税）构成价格天花板
- 饲料需求（尤其生猪）是核心驱动

## LC 碳酸锂

### 定价决定性力量

| 力量 | 说明 |
|------|------|
| **海外矿价** | 澳大利亚锂辉石价格是核心成本 |
| **新能源车产销** | 最大下游，政策补贴/需求增速影响大 |
| **产能周期** | 新增产能投放周期导致供过于求 |
| **库存周期** | 矿石库存/碳酸锂库存是供需的滞后指标 |

### 什么条件下会赢

- **新能源车产销超预期** → 需求好
- **上游矿山减产** → 成本支撑
- **库存去化** → 供需改善

### 历史量价规律

- 2022-2023 年大跌：从 60 万/吨跌至 8 万/吨，是产能过剩的典型案例
- 波动率极大，是高风险品种
- 成交额高的月份合约是主战场

## 通用：容易被操纵的品种特征

满足以下条件的品种容易被资金操纵，分析时需要额外谨慎：

| 特征 | 说明 |
|------|------|
| 成交额排名后 20% | 流动性不足，大单可以左右价格 |
| 持仓量极低 | 逼仓成本低，容易出现极端行情 |
| 合约非主力月份 | 主力合约流动性好，非主力容易被操纵 |
| 现货市场规模小 | 现货价格不透明，期货定价容易被扭曲 |
| 季节性供需错配 | 特定时点供需极不平衡 |

**实际操作原则**：成交额排名后 20% 的品种，除非有明确的产业链逻辑，否则不参与。
