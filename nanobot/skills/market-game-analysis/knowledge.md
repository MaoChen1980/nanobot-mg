## 1. 黑色系

**定价模式**：内盘博弈型 — 钢厂利润是枢纽，进口政策（澳煤/铁矿）是最大外生变量

**黑色系整体维度**（JM/J/RB/HC/I 共性）：

| 维度 | 黑色系整体 |
|------|-----------|
| **产业链** | 焦煤→焦炭→钢铁；铁矿石进口依赖~80% |
| **上游** | 煤矿（国产+进口）+ 四大矿山（淡水河谷/力拓/BHP/FMG）|
| **下游** | 房地产+基建（RB）+ 制造业/汽车/家电（HC）|
| **中枢力量** | 钢厂利润：决定全链条采购意愿和定价权分配 |
| **季节性** | 3-4 旺季/11-12 冬储/春节前后停工 |
| **核心高频** | 周度产量+库存（找钢/钢联）+ 钢厂利润+ 进口政策 |

**黑色系分化条件**：
- 澳煤政策收紧/放开 → JM 单边，独立性最强
- 巴西/澳洲铁矿发运受阻 → I 独立涨跌
- 钢厂利润快速压缩至负 → JM/J/I 先跌（原料弱于成材）
- 钢厂利润 > 200 → 原料(JM/J/I)支撑强

---

### JM 焦煤

| 维度 | 内容 |
|------|------|
| **上游** | 国产（山西/内蒙）+ 进口（澳~30%，边际定价） |
| **下游** | 焦化厂→钢厂（主导定价）|
| **决定性力量** | 澳煤进口政策 + 钢厂利润 + 山西坑口成本 ~1200-1400 |
| **博弈焦点** | 钢厂联合压价 vs 煤矿挺价；进口煤是外生变量 |
| **高频数据** | 澳洲峰景 CFR + 山西低硫主焦煤价 + 焦化厂开工率 |

**现货定价逻辑**：澳煤 CFr 价格决定边际定价（进口恢复=国产煤失去定价权）；山西坑口成本 ~1200-1400 是底部支撑；钢厂联合压价是常态（买方市场）。
**典型错误**：❌"焦煤因下游需求差而跌" → 真实原因常是澳煤进口恢复

---

### J 焦炭

| 维度 | 内容 |
|------|------|
| **上游** | 焦煤（占比成本 ~80%）|
| **下游** | 钢厂（独家买方，定价权极强）|
| **决定性力量** | 焦化厂利润 + 环保限产 + 钢厂采购价（月度招标）|
| **成本线** | 焦化完全成本 ~1800-2200 |
| **高频数据** | 钢厂焦炭库存可用天数 + 焦化厂开工率 + 环保限产 |

**现货定价逻辑**：两头挤压（焦煤涨成本+钢厂压采购价）；钢厂库存可用天数 >15 天→焦化厂被动降价；环保限产是唯一供给收缩逻辑。
**典型错误**：❌"焦炭涨因需求好" → 通常是焦煤成本推动+环保限产

---

### RB 螺纹钢

| 维度 | 内容 |
|------|------|
| **上游** | 钢厂（高炉/电炉），贸易商蓄水池 |
| **下游** | 房地产（~60%）+ 基建（~20%）+ 制造业 |
| **决定性力量** | 房地产新开工 + 基建投资 + 电炉成本 ~3300（边际支撑）|
| **成本线** | 华东长流程 3400-3600；电炉谷电 ~3300；高炉 ~3400 |
| **季节性** | 3-4 月旺季 / 11-12 冬储 / 春节前后停工 |
| **高频数据** | 周度产量+库存+表需（找钢/钢联）+ 电炉开工率 |

**现货定价逻辑**：需求驱动型品种；房地产新开工是最核心变量；电炉成本是边际支撑位（电炉开工会自动调节供给）。
**典型错误**：❌"RB 跌因供给太多" → 根本原因是需求不行

---

### HC 热卷

| 维度 | 内容 |
|------|------|
| **上游** | 钢厂（同 RB，但卷板产线独立）|
| **下游** | 汽车（~25%）+ 家电 + 基建 + 出口 |
| **决定性力量** | 制造业 PMI + 卷螺差 ~200-300 + 冷轧需求 + 出口 |
| **季节性** | 3-5 月汽车旺季 / 9-10 月基建旺季 |
| **高频数据** | 冷轧开工率 + 汽车产销 + 卷螺差 + 出口订单 |

**现货定价逻辑**：当卷螺差 > 400 → 热卷高估，存在套利回归压力；制造业景气度决定方向。
**关键信号**：汽车产销同比 >10% → 热卷需求强；冷热价差 >500 → 冷轧过剩，热卷承压

### I 铁矿石

| 维度 | 内容 |
|------|------|
| **上游** | 四大矿山（淡水河谷/力拓/BHP/FMG），高度垄断 |
| **下游** | 钢厂（买方，定价权弱）|
| **决定性力量** | 普氏 62% 指数 + 矿山发运量 + 港口库存 ~1.2-1.5 亿吨 |
| **成本线** | 四大矿山 C1 成本 30-40 美元/吨（极低成本，跌不破的只有需求崩塌）|
| **高频数据** | 巴西/澳洲周度发运量 + 港口库存（Mysteel）+ 钢厂进口矿库存可用天数 |

**现货定价逻辑**：四大矿山垄断→卖方市场；港口库存是供需平衡表的最直观体现 >1.5亿吨=过剩；钢厂补库周期（库存可用天数<20天→即将补库）是短期驱动。
**典型错误**：❌"I 跌因需求差" → I 成本极低，跌是钢厂负反馈，涨才是真实需求驱动

---

### SS 不锈钢

| 维度 | 内容 |
|------|------|
| **上游** | 镍（~70%不锈钢用途）+ 铬 + 铁合金 |
| **下游** | 制造业（家电/餐厨/建筑装饰）+ 工业设备 |
| **决定性力量** | LME 镍价 + 佛山/无锡社会库存 >60万吨偏空 + 钢厂联合挺价 |
| **高频数据** | 佛山/无锡两地库存 + LME 镍价 + 钢厂盘价 |

**现货定价逻辑**：镍价是成本锚；库存是供需温度计；钢厂月度盘价指引短期方向。
**典型错误**：❌"不锈钢看房地产" → 不锈钢看镍价+制造业，不是螺纹替代品

## 2. 有色金属

**定价模式**：外部定价型 — LME 主导，美元指数+全球 PMI 是宏观底色

### CU 铜

| 维度 | 内容 |
|------|------|
| **上游** | 智利/秘鲁铜矿（全球最大产区）|
| **下游** | 电力电缆~40% + 新能源~15%（电动车+光伏）+ 建筑+家电 |
| **决定性力量** | LME 铜 + 洋山铜溢价 + 保税区库存 + 全球矿山 TC <30=矿端紧缺 |
| **成本线** | 全球矿山 C1 90%分位 ~7000 美元/吨 |
| **高频数据** | LME 注册仓单 + 保税区库存 + 洋山铜溢价 + TC/RC 加工费 |

**现货定价逻辑**：全球库存（LME+SHFE+保税区）是核心定价指标；洋山铜溢价反映中国进口需求强度；LME 注册仓单 <1 万吨=逼仓风险。
**关键信号**：保税区库存 <10万吨→国内偏紧；TC<30 美元→矿端紧缺

---

### AL 铝

| 维度 | 内容 |
|------|------|
| **上游** | 铝土矿（几内亚为主）+ 氧化铝 + 电力 |
| **下游** | 建筑~30% + 新能源车~15% + 光伏~10% + 包装 |
| **决定性力量** | LME 铝 + 云南电力（枯水期限产）+ 产能天花板 4500 万吨 + 社会库存 |
| **成本线** | 全国加权完全成本 ~17000-18000；云南电力是边际变量 |
| **高频数据** | 社会库存 + 云南电解铝开工率 + 铝棒加工费 >500=下游好 |

**现货定价逻辑**：国内供给有产能天花板上限（4500万吨），需求端新能源车+光伏是边际增量；云南枯水期限产是最大季节性供应收缩。
**关键信号**：社会库存 <60万吨=偏紧，>100万吨=过剩

|------|------|
| **上游** | 海外锌精矿（全球分散）|
| **下游** | 镀锌板~60%（建筑+汽车）|
| **决定性力量** | LME 锌 + 镀锌开工率 + TC 加工费 + 矿山供应 |
| **高频数据** | 镀锌板开工率 + LME 锌库存 + TC |

---

### PB 铅

| 维度 | 内容 |
|------|------|
| **上游** | 铅精矿 + 废铅（再生铅占比~50%）|
| **下游** | 铅酸蓄电池~80%（汽车启动电池+电动车电池）|
| **决定性力量** | LME 铅 + 蓄电池开工率 + 环保限产 + 再生铅成本 ~15000-15500 |
| **季节性** | 冬季换电池旺季 |

---

### NI 镍

| 维度 | 内容 |
|------|------|
| **上游** | 印尼/菲律宾镍矿（印尼政策是核心变量）|
| **下游** | 不锈钢~70% + 电池~15%（硫酸镍）|
| **决定性力量** | LME 镍 + 印尼政策（RKAB 配额/限制 RKEF）+ LME 镍库存 |
| **成本线** | 印尼镍铁成本 ~11000-12000 美元/吨 |
| **高频数据** | LME 镍库存 + 印尼镍矿 RKAB 配额 + 不锈钢开工率 |

**现货定价逻辑**：印尼是最大变量（政策决定镍矿供应）；两个市场：镍铁（不锈钢）vs 精炼镍（交割品+电池）。
**典型错误**：❌"镍看新能源就够了" → 镍~70% 用于不锈钢，电池是边际增量

### SN 锡

| 维度 | 内容 |
|------|------|
| **上游** | 缅甸（已大量消耗）+ 印尼（全球最大出口国）|
| **下游** | 焊料~50%（半导体）+ 光伏焊带 + 化工 |
| **决定性力量** | LME 锡 + 缅甸/印尼矿供应 + 半导体周期 |
| **高频数据** | LME 锡库存 + 锡焊料开工率 + 缅甸佤邦政策 |

---

### AO 氧化铝

| 维度 | 内容 |
|------|------|
| **上游** | 国产铝土矿（非外盘锚定）|
| **下游** | 电解铝冶炼 |
| **决定性力量** | 国内铝土矿供应 + 烧碱成本 + 电解铝产能变化 |

**现货定价逻辑**：国内定价品种，不同于其他有色品种（不走 LME 锚定）；电解铝产能变化是最大需求驱动。

---

### BC 国际铜

| 维度 | 内容 |
|------|------|
| **上游** | 进口阴极铜（保税区）|
| **下游** | 同 CU |
| **决定性力量** | LME 铜 + 保税区库存 + 沪伦比值（跨境套利）|

**现货定价逻辑**：保税区铜通过 BC 直接交割；比 CU 更直接反映 LME 价格和进口成本；沪伦比值决定套利方向。

## 3. 贵金属

### AU 黄金

| 维度 | 内容 |
|------|------|
| **上游** | 金矿开采（全球分散）|
| **下游** | 央行储备~30% + 珠宝~50% + ETF/投资~20% |
| **决定性力量** | 美元实际利率（TIPS）+ 全球央行购金量 + 地缘政治 + COMEX 金 |
| **高频数据** | COMEX 金价 + 美元指数 + TIPS 收益率 + 全球央行月度购金量 |

**现货定价逻辑**：实际利率是核心变量，与金价负相关；央行购金是长期边际多头；人民币汇率决定沪金溢价；无成本支撑概念。
**关键信号**：央行月度购金量 >50 吨→长期多头

### AG 白银

| 维度 | 内容 |
|------|------|
| **上游** | 铅锌铜矿伴生（~70%）+ 银矿 |
| **下游** | 工业~50%（光伏银浆+电子）+ 珠宝+投资 |
| **决定性力量** | COMEX 银 + 金银比 ~80-90 + 光伏银浆需求 + 实际利率 |

**现货定价逻辑**：金银比 >100=白银相对低估；黄金涨+PMI 扩张→白银补涨；黄金跌+PMI 收缩→白银超跌。
**典型错误**：❌"白银是贵金属看避险" → 白银一半是工业金属

## 4. 能源

### SC 原油

| 维度 | 内容 |
|------|------|
| **上游** | OPEC+ 产油国 + 美国页岩油 + 俄罗斯 |
| **下游** | 炼厂→成品油（汽柴煤）+ 化工原料 |
| **决定性力量** | OPEC+ 减产执行率 + 全球需求 + 地缘风险 + EIA 库存 |
| **成本线** | OPEC+ 财政盈亏 ~80 美元/桶；美国页岩油 ~60-70 |
| **高频数据** | EIA 周度库存 + OPEC+ 月度产量 + 布伦特月差 |

**现货定价逻辑**：三大定价锚——布伦特（欧洲基准）、WTI（美国基准）、Dubai（亚洲基准）；月差结构（back=偏多，contango=偏空）反映供需松紧；EIA 库存 >5 年均值=偏空。

### BU 沥青

| 维度 | 内容 |
|------|------|
| **上游** | 原油常减压渣油（与燃料油争原料）|
| **下游** | 公路建设（高速公路/市政道路）|
| **决定性力量** | 原油（成本）+ 公路投资计划 + 季节性（3-5月/9-11月旺季）|
| **成本线** | BU-原油裂解价差 ~200-400 元/吨 |
| **季节性** | 3-5 月/9-11 月施工旺季；6-8 月雨季/12-2 月冬季停工 |

**典型错误**：❌"沥青看原油就行" → 沥青更看季节性和投资计划，原油是成本基座不是驱动

## 5. 化工链

**定价模式**：混合型（原油/煤炭成本传导+自身供需）— 加工利润是最关键中间变量

### L 聚乙烯 / PP 聚丙烯

| 维度 | L 聚乙烯 | PP 聚丙烯 |
|------|---------|-----------|
| **上游** | 原油→石脑油→乙烯→LLDPE | 原油→丙烯→PP |
| **下游** | 农膜+包装+管材 | 塑编+注塑+ BOPP 膜 |
| **决定性力量** | 原油 + 进口成本 ~8000-8500 + 农膜开工率 | 原油 + 粉料成本 ~7000-7500 + 塑编开工率 |
| **季节性** | 3-5 月农膜旺季/9-10 月膜料备货 | 春节后开工旺季/7-8 月淡季 |

**现货定价逻辑**：聚烯烃两油库存 >80万吨=供应压力；进口窗口打开=进口套利压制国内价格

---

### PVC 聚氯乙烯

| 维度 | 内容 |
|------|------|
| **上游** | 电石法（~80%，煤化工路线）+ 乙烯法（~20%）|
| **下游** | 管材/型材（建筑~60%）+ 薄膜+地板 |
| **决定性力量** | 电石价 + 煤炭 + 房地产需求 + 社会库存 >40 万吨偏空 |
| **成本线** | 电石法成本 ~6000-6500（煤涨则抬）|

**现货定价逻辑**：房地产需求是核心方向；电石价格是成本锚；出口窗口打开可缓解国内供应压力

### MA 甲醇

**现货定价逻辑**：一年一季作物，花期（4-5 月）天气决定产量预期，是最大价格驱动；库存去化速度决定年度供需松紧。苹果是 AP 特有的交割品级博弈（非标品容易产生交割争议）。
**典型错误**：❌"苹果看消费" → 最大驱动是花期天气和产量，消费相对稳定

## 13. 新能源

### LC 碳酸锂

| 维度 | 内容 |
|------|------|
| **上游** | 进口~60%（智利 SQM/ALB + 澳大利亚 Pilbara）+ 国产盐湖/锂辉石 |
| **成本曲线** | 盐湖提锂 ~7-9 万元/吨；锂辉石 ~10-12 万元/吨 |
| **下游** | 动力电池（~80%）+ 储能 + 3C 消费电子 |
| **决定性力量** | 海外锂矿拍卖价（Pilbara）+ SMM 电池级碳酸锂 + 新能源车产销 + 库存 |
| **季节性** | 1-3 月淡季 / 9-11 月旺季（年底冲量）|
| **高频数据** | Pilbara 拍卖价 + 新能源车月度产销 + SMM 报价 + 库存 >4万吨=过剩 |

**现货定价逻辑**：成本曲线分层是最核心概念——盐湖成本 ~7-9 万是第一支撑，锂辉石 ~10-12 万是第二支撑；下游接受度（电池厂能否传导）决定价格上限；产能周期（3-4 年扩产滞后）决定长期趋势。
**典型错误**：❌"碳酸锂跌因新能源车销量差" → 8-10 万是成本支撑，成本线下移或需求超差才会跌破
