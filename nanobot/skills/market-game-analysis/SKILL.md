# 市场博弈结构分析

## 核心理念（通用）

**"听过其言，不如观其行。"**

- 成交量、持仓量、价格变化是实力的唯一可靠外显信号。
- 叙事/基本面/政策解读都是噪音——只有行为才是信号。
- 市场无客观真理，只有力量对比。

---

## 通用分析框架（适用于所有市场）

```
看到动作 → 追问谁做的、为什么做、能不能持续 → 判断方向 → 站在实力强的那边
```

### 三问穿透法（通用）

1. **谁在做？** —— 找到背后最大的玩家（国家队/产业资本/聪明钱/散户）
2. **为什么做？** —— 动机是什么（套保/投机/护盘/撤退）
3. **有没有实力持续？** —— 资金够不够，动机强不强

### 量价背离核心信号（通用）

| 信号 | 含义 |
|------|------|
| 持仓量↑ + 价格↑ | 多头主动加仓，上涨趋势健康 |
| 持仓量↑ + 价格↓ | 空头主动加仓，下跌趋势健康（空头控盘） |
| 持仓量↓ + 价格↑ | 多头平仓，价格被动上涨（警惕） |
| 持仓量↓ + 价格↓ | 多头砍仓，空头回补，下跌可能尾声 |
| 成交量暴增 + 价格暴跌 | 多头踩踏 or 空头强力砸盘 |
| 成交量缩 + 价格跌 | 空头控盘，多头不接盘 |

### 博弈结构分析维度（通用）

1. **多空力量对比** —— 净头寸是多还是空，规模多大
2. **席位/玩家行为** —— 谁在加仓、谁在减仓
3. **量价配合** —— 趋势是否有资金追随
4. **持续性判断** —— 动力来自投机还是真实需求

## 期货品种分析（国内商品期货）

### 数据获取

```python
import akshare as ak

# 日线行情（量/价/持仓）
df = ak.futures_zh_daily_sina(symbol='AU0')  # 黄金
df = ak.futures_zh_daily_sina(symbol='RB0')  # 螺纹钢
df = ak.futures_zh_daily_sina(symbol='LC0')  # 碳酸锂

# 多空持仓（席位级别）
df_long = ak.futures_hold_pos_sina(symbol='多单持仓', contract='AU2508', date='20250620')
df_short = ak.futures_hold_pos_sina(symbol='空单持仓', contract='AU2508', date='20250620')
```

### 持仓结构分析步骤

1. **拉取日线** → 看价格趋势 + 成交量 + 持仓量变化
2. **拉取多空持仓** → 计算前10合计多空比
3. **追踪席位变化** → 重点关注增减仓最大的会员
4. **判断多空是否分化** → 同一会员多空态度？
5. **结合量价信号** → 持仓量涨+价格跌=空头控盘；持仓量涨+价格涨=多头控盘

### 关键信号解读

| 信号 | 含义 |
|------|------|
| 持仓量↑ + 价格↑ | 多头主动加仓，上涨趋势健康 |
| 持仓量↑ + 价格↓ | 空头主动加仓，下跌趋势健康（空头控盘） |
| 持仓量↓ + 价格↑ | 多头平仓，价格被动上涨（警惕） |
| 持仓量↓ + 价格↓ | 多头砍仓，空头回补，下跌可能尾声 |
| 成交量暴增 + 价格暴跌 | 多头踩踏 or 空头强力砸盘 |
| 成交量缩 + 价格跌 | 空头控盘，多头不接盘 |

### 多空席位分析维度

- **净头寸** = 多方前10合计 - 空方前10合计
- **同一会员双向态度**：国泰君安同时做多做空？净立场是什么？
- **增减仓方向**：谁在加空？谁在减多？
- **产业资本 vs 金融资本**：期货公司背后是产业客户（套保）还是金融客户（投机）？

### 空头控盘教科书信号（2026年6月案例）

```
纯碱 SA2509：
- 净空 24 万手
- 华泰期货单日加空 6,002 手
- 持仓量创历史新高 + 价格创历史新低
→ 持仓量↑ + 价格↓ = 空头主动加仓 = 空头控盘
```

## A股大盘分析

### 核心框架：谁是大庄

**A股大庄 = 国家队（汇金/社保/证金）**

| 资金 | 特点 |
|------|------|
| 国家队 | 维护稳定，不是盈利，可以无限期持有 |
| 社保基金 | 超长周期，加仓AI算力/光伏/化工 |
| 汇金 | 重仓银行，守金融 |
| 外资（北向）| 最接近聪明钱，持续性有限 |

### 数据获取

```python
# 指数数据（注意：东方财富接口近期不稳定）
df = ak.index_zh_a_hist(symbol='000001', period='daily', start_date='20260501', end_date='20250621')

# 北向资金
df = ak.stock_hsgt_north_net_flow_in_em(symbol='北向资金')  # 接口名可能有变

# 板块涨跌
df = ak.stock_board_industry_name_em()  # 接口不稳定

# 融资融券
df = ak.stock_margin_detail_sse()  # 沪市融资融券
```

### 大盘博弈结构分析步骤

1. **国家队持仓动向** —— 一季度总持仓是否扩张/收缩？（4.9万亿→4.4万亿=收缩=不全面护盘）
2. **指数 vs 个股** —— 指数涨但个股跌=拉指数行情，非真实做多
3. **成交额变化** —— 缩量=观望；放量=方向选择临近
4. **板块分化** —— 谁在领涨/领跌，反映机构调仓方向

### A股特有分析维度

| 维度 | 含义 |
|------|------|
| 国家队是否入场 | 最大庄的动作 |
| 政策方向 | 国家想让哪个方向涨 |
| 北向资金持续性 | 外资是短期还是长期 |
| 机构调仓方向 | 从红利股切向科技？ |
| 涨跌停家数 | 情绪极端值 |

## A股板块分析

### 追踪方向

1. **政策支持方向** —— 国家队/社保在加什么板块
2. **板块量价配合** —— 有没有资金持续流入
3. **机构vs游资** —— 机构在买的板块有持续性，游资题材股容易一日游

### 板块机会判断

| 场景 | 信号 |
|------|------|
| 有机会 | 国家队/社保已在加仓 + 估值合理 + 政策支持 |
| 没有机会 | 国家队总持仓收缩 + 没有明确主线 + 成交额缩量 |
| 等待信号 | 红利股补跌完 + 外资持续流入 + 成交额重回4万亿 |

## 美股博弈分析

### 数据获取

```python
# yfinance（注意：有速率限制，同一IP短时间多次调用会报 YFRateLimitError）
import yfinance as yf

nvda = yf.Ticker('NVDA')

# 基本面（有空信息会报错，用 .get() 兜底）
info = nvda.info
pe = info.get('forwardPE')
short_float = info.get('shortPercentOfFloat')  # 做空比例
borrow_rate = info.get('borrowRate')          # 借券费率

# 机构持仓
df = nvda.institutional_holders  # 13F，季度披露，滞后

# 期权链（触发速率限制）
opts = nvda.options              # 到期日列表
chain = nvda.option_chain(opts[0])  # 指定到期日的完整链

# PCR计算
total_call_vol = chain.calls['volume'].sum()
total_put_vol = chain.puts['volume'].sum()
pcr_vol = total_put_vol / total_call_vol

total_call_oi = chain.calls['openInterest'].sum()
total_put_oi = chain.puts['openInterest'].sum()
pcr_oi = total_put_oi / total_call_oi
```

### 关键博弈指标

| 指标 | 获取方式 | 含义 |
|------|---------|------|
| Short Interest / Float | `info['shortPercentOfFloat']` | 做空比例，高位=看空情绪浓 |
| Borrow Rate | `info['borrowRate']` | 借券费率，高=做空成本高，空头不愿留 |
| Put/Call Volume Ratio | 期权链计算 | >1.2空头情绪，<0.7多头情绪 |
| Put/Call Open Interest | 期权链计算 | 持仓量层面的多空对比 |
| IV Rank | `optioncharts.io` 或 barchart | 波动率历史分位，高IV=期权贵 |
| Unusual Options Activity | barchart / fintel | 成交量远超OI=机构大单 |

### 量价分析（美股ETF优先）

```python
# ETF资金流是最接近"席位数据"的美股代理
spy = yf.Ticker('SPY')
qqq = yf.Ticker('QQQ')
# 注意：yfinance不直接提供ETF申购赎回，用价格+成交量代替

# 日线
df = yf.download('NVDA', period='1mo', interval='1d')
# 持仓量 → 免费提供（yfinance的日线数据包含Volume）
```

### 博弈解读框架

| 信号组合 | 市场含义 |
|---------|---------|
| 借券费率升高 + 做空比例下降 | 空头被迫平仓，反弹信号 |
| PCR升高 + 价格上涨 | 机构用期权对冲，看涨但谨慎 |
| PCR下降 + 价格上涨 | 上涨无阻力，看涨情绪乐观 |
| IV Rank > 70 + 价格高位 | 期权溢价贵，机构在买保险=警惕 |
| Unusual Call Activity + 价格 | 机构在赌方向向上 |
| Unusual Put Activity + 价格 | 机构在赌方向向下或对冲 |

### 重要坑点

1. **yfinance 速率限制**：触发后等待5-10分钟，同一IP多次调用会封
2. **期权数据滞后**：otm期权链次日才更新
3. **13F季度滞后**：机构持仓披露滞后一个季度
4. **Web数据获取替代**：barchart/fintel有反爬，用web_search/web_fetch补充

## 机会排序方法（通用公式）

### 期货品种排序

```
做空机会清晰度 = 空头净持仓规模 × 空头加仓力度 × 量价配合程度
做多机会清晰度 = 多头净持仓规模 × 多头加仓力度 × 量价配合程度
```

### A股板块排序

```
机会强度 = 国家队/社保持仓方向 × 政策支持力度 × 估值合理性 × 资金流入持续性
```

## 期权 Put/Call 比分析

### 数据获取

```python
# 50ETF 期权波动率指数（VIX）
df = ak.index_option_50etf_qvix()  # 注意：有bug，pandas版本问题

# 沪深300期权（最活跃）
df = ak.option_cffex_hs300_spot_sina()  # 买量/卖量/持仓量/最新价

# 商品期权隐含波动率（目前数据为空，慎用）
ak.option_vol_shfe(symbol='AU')  # 黄金
ak.option_vol_shfe(symbol='CU')  # 铜
```

### Put/Call 比计算

```
PCR (Put/Call Ratio) = Put持仓量 / Call持仓量

PCR > 1.2 → 看跌情绪浓
PCR < 0.7 → 看涨情绪浓
```

### 博弈解读

| 信号 | 含义 |
|------|------|
| PCR升高 + 价格上涨 | 多头借助期权对冲风险，看涨但谨慎 |
| PCR升高 + 价格下跌 | 空头主导，市场恐慌 |
| PCR下降 + 价格上涨 | 上涨无阻力，看涨情绪乐观 |
| PCR下降 + 价格下跌 | 上涨乏力，空头开始建仓 |

### 注意

- **akshare期权数据不稳定**：50ETF波动率有pandas类型bug；商品期权波动率返回空数据
- **数据为空时**：用 `web_search` / `web_fetch` 补充，或直接用期货席位持仓代替

## 风险控制原则

1. **没有数据不分析** —— 接口故障时用 web_search/web_fetch 补充
2. **没有清晰信号不行动** —— 等国家队/聪明钱给出方向再跟
3. **杠杆产品严格止损** —— 期货带杠杆，止损是生命线
4. **区分"有机会"和"该行动"** —— 有机会不代表现在该进，要等信号确认
5. **持续追踪信号** —— 每日持仓结构变化比单次数据更重要

---

## 常见误区（避免这些错误）

| 误区 | 正确做法 |
|------|---------|
| "超跌=要反弹" | 超跌只是叙事，反弹需要资金接盘 |
| "国家队护盘=会涨" | 国家队护的是稳定，不是帮你赚钱 |
| "PE低=估值底" | 国家队在减仓时，PE再低也可以继续跌 |
| "技术指标金叉=买入信号" | 技术指标研究的是价格规律，不是人 |
| "利好=要涨" | 利好是讲故事，机构借机出货才是真 |
