# MGA — Market Game Analysis

## 核心理念

**穿透现象，看清本质。**

价格/量/OI/席位都是现象。MGA的目标是通过这些现象，还原背后多空博弈方的动机、实力和意图。

**三问穿透法是MGA的灵魂**（必须先于任何工具调用）：
1. **谁在打谁** — 多空身份、动机，成本位
2. **打在哪个价位** — 关键支撑/阻力在哪里
3. **谁能打赢** — 实力对比，席位+OI+仓单

---

## ⚠️ 数据时效分层规范（MGA分析第一步）

**核心原则：不同数据源的时间戳必须严格分离，禁止混用。**

### 三层时间分层

```
┌──────────────────────────────────────────────────────────────┐
│ Layer 1: 实时快讯（当日盘中）                                │
│  - 来源：东财快讯 qhweb.eastmoney.com/kuaixun               │
│  - 时间戳：当日 09:00 - 13:xx（盘中实时）                    │
│  - 用途：判断"今天正在发生什么"，涨跌幅为盘中实时信号         │
│  - 注意：只有涨跌幅和品种强弱，无精确价位（除非快讯内含）     │
│  - 置信度：高（当日真实行情）                                │
├──────────────────────────────────────────────────────────────┤
│ Layer 2: 昨日收盘价（akshare futures_zh_daily_sina）         │
│  - 来源：akshare futures_zh_daily_sina(symbol="CU2607")    │
│  - 时间戳：前一交易日收盘（如 2026-06-24，当系统时间为       │
│    2026-06-25 盘中时）                                       │
│  - 用途：计算精确涨跌% + 持仓量变化                           │
│  - 陷阱：系统时间≠数据时间，akshare 数据永远比"现在"晚1天    │
│  - 置信度：中（历史数据，需结合快讯判断今日是否加速）        │
├──────────────────────────────────────────────────────────────┤
│ Layer 3: T-1 席位数据（SHFE/DCE 席位排名）                   │
│  - 来源：akshare get_shfe_rank_table(date="2026-06-24")     │
│  - 时间戳：T-1 报告（席位数据比行情晚1天）                   │
│  - 用途：验证"谁在打"——多空席位集中度、净持仓方向           │
│  - 置信度：中（滞后1天，但能穿透多空身份）                   │
└──────────────────────────────────────────────────────────────┘
```

### ⚠️ 致命陷阱：昨收≠今行情（2026-06-25 实测验证）

```
错误用法：
  akshare 沪铜 CU2607 昨收 = 103520
  → 误判："沪铜只跌了 -0.06%，表现抗跌"
  → 错！因为：东财快讯显示沪铜夜盘跳空 -1.28%，早盘跌超1%

正确用法：
  Layer 2（akshare昨收）：103520（-0.06%）→ 昨日收盘基本持平
  Layer 1（东财快讯）：沪铜夜盘跳空-1.28%，早盘续跌 → 今日在加速
  结论：昨收"抗跌"是假象，实际空头已在夜盘建仓
```

### 加速信号识别（关键战法）

当 Layer1（快讯跌幅）>> Layer2（昨收跌幅）时，行情正在加速：

| 品种 | 06-24 昨收涨跌 | 06-25 盘中涨跌 | 加速倍数 | 信号 |
|------|--------------|--------------|---------|------|
| PTA | -1.80% | -6% | **3.3×** | 空头加速建仓 |
| 橡胶 | -0.40% | -7% | **17.5×** | 崩跌，不是正常回调 |
| 沪银 | -0.31% | -7% | **22×** | 流动性踩踏 |
| 沪金 | -0.21% | -2% | **9.5×** | 虽加速但幅度可控 |

**操作含义**：加速信号出现 → 原趋势动能强 → 持空/加空，不轻易抄底

## 盘中参考价估算公式（昨收 ± 盘中涨跌）

**核心原则**：akshare 的 `futures_zh_daily_sina` 只能拉最近交易日收盘（L2昨收），不等于今行情。盘中涨跌须从东财快讯获取（L1盘中），两套数据严格分层，绝不可混用。

### 估算公式

```
今日估算价位 = 06-24收盘价 × (1 + 06-25盘中涨跌%)
```

**⚠️ 重要区分**：
- 快讯提供「精确价位」（如"5396"）→ 直接使用精确价位
- 快讯提供「文字描述」（如"跌超5%"）→ 取区间中值估算

### 实战计算（2026-06-25）

| 品种 | 06-24收盘 | 06-25盘中涨跌 | 今日估算价位 | 快讯精确价位 | 偏差 |
|------|---------|------------|------------|------------|------|
| PTA | 5672 | -6% | 5332 | 5396（午盘） | +64 |
| 橡胶 | 17605 | -7% | 16373 | 14505（低见） | +1868 |
| 沪银 | 14950 | -7% | 13904 | — | — |
| 沪金 | 896.04 | -2% | 878.1 | — | — |
| 铁矿石 | 744 | +0.5% | 747.7 | — | — |

### PVC 空头减仓反弹陷阱（2026-06-25 实测）

| 信号 | 数据 | 解读 |
|------|------|------|
| 持仓量变化 | **-4.4%**（持仓大降） | 空头在平仓 |
| 价格 | **+1.04%**（价格上涨） | 多头没进场，是空头撤退 |
| 综合判断 | **空头减仓，非真涨** | 反弹即放空机会 |

**判断口诀**：持仓降 + 价格涨 = 空头撤退（不追多）；持仓降 + 价格跌 = 多头踩踏（不抄底）

## 情绪维度 — 成交额信号

成交额反映市场情绪温度，是 MGA 分析的辅助维度（不占席位/OI核心权重）。

### 成交额信号解读

| 信号 | 解读 | 操作提示 |
|------|------|---------|
| 连续放量（>2万亿/日） | 市场活跃，趋势延续概率上升 | 顺势 |
| 单日放量 >3000亿 | 情绪过热，短期回调风险 | 减仓/止损收紧 |
| 持续缩量（<8000亿） | 观望浓厚，震荡格局 | 减少操作 |
| 成交额与指数背离 | 权重护盘，非真强势 | 谨慎 |

### 实战案例（2026-06-25）

> A股成交额 2.5万亿，连续43个交易日突破2.5万亿，较上日**放量3214亿**

- **归类**：情绪过热信号（单日放量3214亿 > 3000亿阈值）
- **权重板块**：港股中资银行（浙商/中行/工行等）领跌 -6%，传导A股权重
- **操作含义**：情绪过热 + 权重补跌 = 大盘短期回调风险上升，降低大盘仓位

港股 / A股 / 商品期货存在领先-滞后关系，联动分析可提升方向判断置信度。

| 关联路径 | 信号类型 | 联动时滞 | 实战案例（2026-06-25） |
|----------|---------|---------|----------------------|
| 港股中资银行 → A股权重银行 | 先行指标 | 5-30分钟 | 港股中行-6% → A股银行跟跌 |

```
□ 港股银行板块今日涨跌？（先行信号）
```

## 三问穿透法（详细版）

### Q1: 谁在打谁

**穿透什么**：多空双方的身份、动机、成本位

| 信号 | 穿透结论 |
|------|---------|
| 席位净空 + OI增 | 空头主力建仓，席位数据验证 |
| 席位净多 + OI减 | 多头止盈，非空头打压 |
| OI增 + 价格涨 | 多头建仓 |
| OI减 + 价格跌 | 空头平仓（非新空进入）|
| 仓单增 + 价格跌 | 产业套保卖盘，压制价格 |
| 仓单减 + 价格涨 | 产业买保，支撑价格 |
| **持仓量骤降 + 价格加速下跌** | 多头踩踏/空头平仓，非基本面驱动，需谨慎 |

**席位数据优先级**（席位是验证"谁"的核心）：
- 净空集中度高 → 空头主力明确
- 净多集中度高 → 多头主力明确
- 席位分散 → 散户行情，非主力行为

### Q2: 打在哪个价位

**穿透什么**：关键支撑/阻力在哪里，多空的成本位和目标位

**⚠️ 盘中价位判断必须区分"昨收价位"和"实时价位"：**

| 价位类型 | 定义 | 意义 |
|---------|------|------|
| 结构支撑 | 10日/20日/40日低点 | 多头防线 |
| 结构阻力 | 10日/20日/40日高点 | 空头防线 |
| 整数关口 | 15000/14000/10000等 | 心理位置 |
| 前高/前低 | 历史极值 | 趋势确认位 |
| 持仓密集区 | OI堆积价位 | 止损密集区 |

**关键判断**：价格跌破支撑后，如果OI不增，说明空头没认真打，可能是假跌破。

**加速突破判断**：快讯盘中跌幅 > 昨收跌幅 × 3倍 → 行情正在加速 → 不抄底

### Q3: 谁能打赢

**穿透什么**：多空双方实力对比

| 实力指标 | 强空头 | 强多头 |
|---------|-------|-------|
| 席位 | 净空大 + 席位集中 | 净多大 + 席位集中 |
| OI | 持续增仓 | 持续增仓 |
| 仓单 | 仓单增（产业卖保） | 仓单减（产业买保） |
| 基差 | 贴水加深 | 升水扩大 |
| 价格（加速信号）| 盘中跌幅 >> 昨收跌幅 | 盘中涨幅 >> 昨收涨幅 |

**⚠️ 持仓量下降 + 价格加速 → 空头陷阱或流动性踩踏**
（PVC持仓降-4.4% + 价格涨1.04% = 空头减仓反弹，非真涨）

## 四维确认框架

四维是验证三问的数据工具，不是目的。

| 维度 | 数据来源 | 通过条件 |
|------|---------|---------|
| **L1趋势** | MA20/MA40方向（注：商品期货可调整为10/20日线，MA20在MA40同侧 + 价格在MA40同向） | 价格在MA40同侧，MA20在MA40同向 |
| **L2资金** | 3日OI轨迹 | OI和价格同向变化（⚠️ 注意：akshare `hold`=期货公司保证金口径，≠ 交易所持仓量OI，混用会导致判断错误） |
| **L3结构** | 10日/20日高低点 | 突破/跌破前高前低 |
| **L4风控** | ATR止损 | 止损=入场价±N×ATR（N由品种特性决定，参考值1.5） |

**四维评分计算规则**：

每维满分 3 分，总分 12 分。

| 维度 | 通过条件 | 得分 |
|------|---------|------|
| **L1趋势** | MA20在MA40同侧 + 价格在MA40同侧 | 3/3 |
|  | MA20在MA40同侧，但价格在MA40另一侧 | 1/3 |
|  | MA20与MA40反向 | 0/3 |
| **L2资金** | 3日OI轨迹与价格完全同向 | 3/3 |
|  | OI增但价格不涨，或OI减但价格不跌 | 1/3 |
|  | OI与价格完全反向 | 0/3 |
| **L3结构** | 突破前10日/20日高低点 | 3/3 |
|  | 触及结构位但未突破（上下影线） | 1/3 |
|  | 在结构位内震荡 | 0/3 |
| **L4风控** | 止损距离 ≤ 1.0×ATR | 3/3 |
|  | 止损距离 1.0-2.0×ATR | 2/3 |
|  | 止损距离 2.0-3.0×ATR | 1/3 |
|  | 止损距离 > 3.0×ATR（不划算） | 0/3 |

**置信度计算**：

```
框架置信度分 = 四维总分 / 12 × 100%
（例：四维得 9/12 → 框架置信度分 = 9÷12×100% = 75%）

数据可用性系数 = 见下方置信度体系表格
（例：席位+OI双确认 → 系数 = 1.0）

最终置信度 = 框架置信度分 × 数据可用性系数
（例：75% × 1.0 = 75%，即"高"置信度）
```

**四维评分决策阈值**（由品种特性决定，以下为示例）：

| 四维总分 | 含义 | 决策 |
|------|------|------|
| 总分 ≥ 总分上限×75% | 四维全共振 | 可重仓（仓位系数 1.0） |
| 总分 ≥ 总分上限×58% | 三维确认 | 标准仓位（系数 0.8） |
| 总分 ≥ 总分上限×42% | 两维确认 | 轻仓/观望（系数 0.5） |
| 总分 < 总分上限×42% | 信号弱 | 不做 |

> 示例基于12分总分。高波动品种（如原油）可上调阈值，低波动品种（如贵金属）可下调。

## 决策规则

### 入场规则

```
IF 三问假设已形成
AND 四维评分 ≥ 7/12
AND 有明确的止损位
AND 席位（若可用则验证，不可用则降档后入场）
AND 已确认各数据源的时间戳（昨收/盘中/席位T-1）
THEN → 入场
```

### 止损规则

| 信号 | 操作 |
|------|------|
| 价格反向突破结构位 | 止损 |
| OI突然大降（>10%）| 先减仓，确认后再处理 |
| 新信号反向（同一品种）| 先减仓，不反手 |
| 席位净持仓突变 | 重新三问穿透 |
| **持仓量降 + 盘中加速**（快讯跌幅>>昨收跌幅）| 谨慎，可能是流动性踩踏 |

### 盘中应对

| 信号 | 解读 | 操作 |
|------|------|------|
| OI增 + 价格不涨 | 空头建仓/多头护盘 | 观察 |
| OI减 + 价格不跌 | 多头平仓/空头护盘 | 观察 |
| OI增 + 价格涨 | 多头建仓 | 持多/跟多 |
| OI减 + 价格跌 | 空头平仓 | 不追 |
| **快讯盘中跌幅 >> 昨收跌幅（×3倍+）** | 行情加速，不逆势 | 持空/加空，不抄底 |

### 仓位管理

```
单笔仓位上限 = 账户净值 × 2%
品种持仓上限 = 账户净值 × 5%
总持仓上限 = 账户净值 × 15%

仓位计算：
  实际手数 = 账户净值 × 仓位系数 / (价格 × 每手吨数 × 保证金率)

**典型品种保证金率参考**（akshare futures_zh_daily_sina 返回字段含 `trade_date`/`vol`/`hold`，保证金率查期货公司官网或经纪商数据）：

| 品种 | 合约 | 保证金率（主力合约） |
|------|------|------------------|
| 螺纹钢 RB | RB2609 | ~9% |
| 铁矿石 I | I2609 | ~12% |
| 焦煤 JM | JM2609 | ~12% |
| 焦炭 J | J2609 | ~12% |
| 沪金 AU | AU2608 | ~8% |
| 沪银 AG | AG2608 | ~10% |
| 沪铜 CU | CU2607 | ~7% |
| 橡胶 RU | RU2609 | ~10% |
| PTA TA | TA2609 | ~8% |

**ATR止损参数N参考值**（由品种波动率决定，使用者可根据历史ATR自设）：

| 波动率 | 品种举例 | 推荐N |
|--------|---------|------|
| 低波动 | 黄金、白银 | 1.5 |
| 中波动 | 螺纹钢、焦煤 | 1.5–2.0 |
| 高波动 | 原油、橡胶、铁矿石 | 2.0–2.5 |
```

| 仓位系数 | 来源 | 说明 |
|---------|------|------|
| 1.0 | 四维评分 ≥ 9/12 | 可重仓 |
| 0.8 | 四维评分 7-8/12 | 标准仓位 |
| 0.5 | 四维评分 5-6/12 | 轻仓 |
| 0 | 四维评分 < 5/12 | 不做 |

### 主动退出规则

在以下条件满足时，**主动平仓**（不等止损位触发）：

| 条件 | 触发操作 | 理由 |
|------|---------|------|
| 浮盈达到入场价的 **3×止损幅度** | 全部平仓 | 盈亏比到位，锁定利润 |
| 价格在目标价位遇到**强阻力**（OI增+价格不涨）| 先减仓50% | 可能有空头护盘 |
| 持仓超过 **3个交易日** 且无浮盈 | 全部平仓 | 时间成本过高 |
| 浮盈回撤超过 **50%**（从最高点）| 全部平仓 | 趋势可能反转 |
| 归因驱动因素消失（新闻反转/政策叫停）| 全部平仓 | 持有逻辑已变 |

### 信号特征库

按品种归档：
- 什么情况下该品种信号胜率高？
- 什么情况下该品种信号经常失效？
- 失效前兆是什么？
- 加速信号（盘中>>昨收）在该品种上的胜率？

### 复盘结论落地机制

复盘后，**信号特征库的结论必须反馈到下次分析**，具体规则：

| 复盘结论 | 落地操作 |
|---------|---------|
| 某品种"OI增+价格涨"信号历史胜率 > 70% | 下次遇到同样信号，提高L2资金维度得分上限 |
| 某品种"触及结构位但不突破"经常失效 | 下次遇到结构位，评分降一档 |
| 归因强的信号胜率显著高于归因弱的 | 提高归因置信度权重 |
| 加速信号（盘中>>昨收×3）有效预测趋势延续 | 下次遇到加速信号，提高持仓决心 |
| 连续3次止损同类型信号 | 暂停该类型信号，复查框架是否失效 |

**复盘→框架迭代的闭环路径**：

```
每10笔交易 → 统计胜率 → 更新置信度系数基准
每20笔交易 → 分析失效特征 → 调整四维权重或评分规则
每50笔交易 → 全面复盘 → 决定是否修订框架本身
```
